# SmartBB Client Code

The SmartBB client is a Bluetooth enabled child lock device based around the [BlueNRG-2](https://www.st.com/en/wireless-transceivers-mcus-and-modules/bluenrg-2.html) microcontroller from STMicroelectronics.

## Installation

A BlueNRG-2 example project to measure power consumption of the BlueNRG-2 chip was used as a base for the client code. This was because the example project had the BlueNRG-2 chip advertise itself and accept any connections. A raspberry Pi was able to successfully connect to the chip running this example project.

This example project, and the client code was/is being developed in TrueStudio IDE. In the `\Client Code` directory in this repo, the metadata and project files for the Client Code TrueStudio Project are provided, as well as a zipped version of the two.

Opening this project and building will result in no errors and a flashable image.
